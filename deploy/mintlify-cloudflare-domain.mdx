# <img src="/icons/domain.svg" alt="Domain icon" width="22" /> Mintlify + Cloudflare Custom Domain

Publish documentation at `docs.aicord.cloud`.

## Steps

1. Connect repository in Mintlify dashboard
2. Configure custom domain: `docs.aicord.cloud`
3. In Cloudflare DNS, create:
   - Type: `CNAME`
   - Name: `docs`
   - Target: `cname.mintlify-dns.com`
4. Set SSL/TLS to `Full (strict)` after certificate provisioning

## Validation checklist

- [ ] `https://docs.aicord.cloud` resolves
- [ ] TLS certificate is valid
- [ ] No mixed-content warnings
- [ ] Mintlify build deploys from Git pushes
