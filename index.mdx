## Production baseline

- Target infra profile: CPU-only hybrid (local + remote fallback)
- Public safe capacity target: 10-20 concurrent users at launch
- SLO target:
  - `p95 <= 4s` for short requests
  - `p95 <= 8s` for complex requests
