# 🛡️ Safety and Security

> Security baseline for selling and operating AiCord in production.

## 🔐 Key handling

- Never commit API keys to Git
- Use secret manager for production
- Rotate tenant keys periodically

## 👮 Access control

- Enforce least privilege on panel/server access
- Restrict management commands to owner/admin IDs

## ⚙️ Runtime guardrails

- Request timeouts
- Per-tenant rate limits
- Queue + backpressure controls

## 📜 Logging policy

- Log status code, latency, tier, upstream route
- Never log full secrets
- Keep timeline data for postmortems

## 📦 Licensing and distribution

- Keep Paid support scope explicit
- Keep trial restrictions documented
