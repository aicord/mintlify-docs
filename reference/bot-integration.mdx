# 🤖 Bot Integration (AiCord -> AiCordCloud)

> Remove provider-key friction for your customers.

## ⚙️ Bot `.env`

```env
AI_PROVIDER=openai
OPENAI_BASE_URL=https://api.aicord.cloud/v1
OPENAI_API_KEY=<tenant_key>
OPENAI_MODEL=aicord/fast
AI_FALLBACK_PROVIDERS=
```

## 🧭 Routing notes

- Use `aicord/fast` as default low-latency tier
- Use `aicord/balanced` for heavier code tasks
- Use `aicord/best` for premium flows only

## ✅ Validation checklist

- [ ] Bot starts with no provider auth errors
- [ ] Test prompt returns successful response
- [ ] Invalid key logs explicit `401`
- [ ] Rate-limited requests return `429` + retry path

## 🛡️ Recommended guardrails

- Strict request timeouts
- Max token limits in user channels
- Queue/backpressure on API gateway
