# <img src="/icons/api.svg" alt="API icon" width="22" /> API Reference (AiCordCloud)

Base URL:

```text
https://api.aicord.cloud
```

## <img src="/icons/key.svg" alt="Key icon" width="18" /> Authentication

Use one of:

- `Authorization: Bearer <API_KEY>`
- `x-api-key: <API_KEY>`

## <img src="/icons/reference.svg" alt="Reference icon" width="18" /> Endpoints

| Method | Path | Description |
| --- | --- | --- |
| `GET` | `/` | Service metadata |
| `GET` | `/health` | Health signal |
| `GET` | `/v1/models` | Public model aliases |
| `POST` | `/v1/chat/completions` | Chat completion |

### `GET /health` response

```json
{"ok": true}
```

### `POST /v1/chat/completions` example

```bash
curl https://api.aicord.cloud/v1/chat/completions \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "aicord/fast",
    "messages": [
      {"role":"system","content":"You are a coding assistant."},
      {"role":"user","content":"Create a Discord slash command /ping."}
    ],
    "temperature": 0.2
  }'
```

## <img src="/icons/process.svg" alt="Headers icon" width="18" /> Response headers

- `x-aicord-tier`
- `x-aicord-model`
- `x-aicord-upstream`
- `x-aicord-latency-ms`
- `x-ratelimit-remaining`

## <img src="/icons/troubleshoot.svg" alt="Error icon" width="18" /> Error format

```json
{
  "error": {
    "message": "Unauthorized",
    "type": "authentication_error"
  }
}
```
