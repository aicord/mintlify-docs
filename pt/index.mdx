## Perfil de producao

- Infra alvo: CPU-only hibrido (local + fallback remoto)
- Meta segura de lancamento: 10-20 simultaneos
- SLO:
  - `p95 <= 4s` para requests curtos
  - `p95 <= 8s` para requests complexos
