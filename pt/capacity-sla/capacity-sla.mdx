# 📈 Capacity and SLA (PT-BR)

> Metas conservadoras para lancamento com qualidade.

## 🧱 Perfil base

- 8 vCPU
- 24 GB RAM
- NVMe
- CPU-only hibrido

## 👥 Capacidade segura

- 10-20 simultaneos

## ⏱️ SLO

- `p95 <= 4s` para requests curtos
- `p95 <= 8s` para requests complexos

## 🔬 Fatores de performance

- Tamanho do prompt
- Limite de tokens
- Perfil do modelo local
- Profundidade de fila e saude do upstream

## 📊 Politica de escala

Quando passar da faixa segura:

1. Reforcar fila e limites
2. Mover cargas pesadas para remoto
3. Escalar instancias ou migrar inferencia local para GPU

## 💼 Regra comercial

Nao prometer concorrencia ilimitada.
Publicar faixa segura e monitorar telemetria real.
