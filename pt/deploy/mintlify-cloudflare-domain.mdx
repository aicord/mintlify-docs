# Dominio customizado com Mintlify + Cloudflare

Esta pagina cobre a publicacao de `docs.aicord.cloud`.

## 1. Conectar repositorio no Mintlify

1. Abra o dashboard Mintlify.
2. Conecte o repositorio GitHub.
3. Confirme o uso de `docs/docs.json`.

## 2. Configurar dominio customizado

No painel Mintlify, configure:

- `docs.aicord.cloud`

## 3. Configurar DNS no Cloudflare

Criar registro:

- Tipo: `CNAME`
- Nome: `docs`
- Destino: `cname.mintlify-dns.com`
- Proxy: seguir recomendacao Mintlify (normalmente DNS only durante validacao)

## 4. SSL/TLS

No Cloudflare SSL/TLS:

- Usar `Full (strict)` apos provisionamento de certificado.

## 5. Checklist de validacao

1. `https://docs.aicord.cloud` responde.
2. Certificado valido.
3. Sem warning de mixed content.
4. Build Mintlify conectado a push Git.

## 6. Operacao continua

- Documentar registros DNS.
- Revalidar dominio apos mudancas de DNS/seguranca.
