# 🌐 Dominio customizado com Mintlify + Cloudflare

> Publicacao da docs em `docs.aicord.cloud`.

## 1. 🔗 Conectar repositorio no Mintlify

1. Abrir dashboard Mintlify
2. Conectar repositorio GitHub
3. Confirmar `docs/docs.json` como configuracao principal

## 2. 🏷️ Configurar dominio

Definir dominio:

- `docs.aicord.cloud`

## 3. ☁️ Configurar DNS no Cloudflare

Criar registro:

- Tipo: `CNAME`
- Nome: `docs`
- Destino: `cname.mintlify-dns.com`
- Proxy: seguir recomendacao Mintlify durante validacao

## 4. 🔒 SSL/TLS

Definir modo:

- `Full (strict)` apos provisionamento de certificado

## 5. ✅ Checklist de validacao

- [ ] `https://docs.aicord.cloud` responde
- [ ] Certificado valido
- [ ] Sem mixed-content warning
- [ ] Build Mintlify conectado ao Git

## 6. 🔁 Operacao continua

- Manter DNS documentado
- Revalidar dominio apos mudancas de rede/seguranca
