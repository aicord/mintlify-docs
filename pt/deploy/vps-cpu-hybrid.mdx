# 🖥️ Deploy VPS CPU Hybrid (PT-BR)

> Perfil de lancamento: **8 vCPU · 24 GB RAM · NVMe · CPU-only hibrido**

## 🏗️ Arquitetura

1. Bot chama `api.aicord.cloud`
2. `aicord/fast` vai para modelo local em CPU
3. Requests pesados/pico vao para fallback remoto

## ⚙️ Baseline recomendado

```env
REQUEST_TIMEOUT_MS=12000
RATE_LIMIT_PER_MINUTE=120
ALLOW_DIRECT_MODEL=false
UPSTREAM_OVERRIDE_MODEL=qwen2.5-coder:7b
```

## 🚀 Passos de deploy

1. Provisionar VPS e atualizar pacotes
2. Instalar Python 3.11+ e criar venv
3. Configurar `AICORDCLOUD/.env`
4. Subir gateway (`python run.py` ou gerenciador de processo)
5. Colocar reverse proxy com TLS
6. Apontar DNS de `api.aicord.cloud`

## 🔐 DNS + TLS

- Host API: `api.aicord.cloud`
- Host Docs: `docs.aicord.cloud`
- Usar TLS estrito com renovacao automatica

## ✅ Verificacao pos deploy

- [ ] `GET /health` retorna `{"ok": true}`
- [ ] `GET /v1/models` lista aliases esperados
- [ ] `POST /v1/chat/completions` funciona com chave valida
- [ ] Chave invalida retorna `401`
- [ ] Rate limit retorna `429`
