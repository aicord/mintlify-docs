# 🤖 Integracao com o Bot (PT-BR)

> Remova friccao de chave externa para seu cliente final.

## ⚙️ `.env` do bot

```env
AI_PROVIDER=openai
OPENAI_BASE_URL=https://api.aicord.cloud/v1
OPENAI_API_KEY=<tenant_key>
OPENAI_MODEL=aicord/fast
AI_FALLBACK_PROVIDERS=
```

## 🧭 Regras de roteamento

- `aicord/fast` como padrao de baixa latencia
- `aicord/balanced` para tarefas de codigo mais pesadas
- `aicord/best` para fluxo premium

## ✅ Checklist de validacao

- [ ] Bot inicia sem erro de autenticacao de provider
- [ ] Prompt de teste retorna com sucesso
- [ ] Chave invalida gera `401` explicito
- [ ] Limite de taxa gera `429` + estrategia de retry

## 🛡️ Guardrails recomendados

- Timeout estrito
- Limite de tokens por canal de usuario
- Fila/backpressure no gateway
