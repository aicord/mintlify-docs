# 📡 Referencia de API (PT-BR)

> Endpoints oficiais do gateway AiCordCloud (compatibilidade OpenAI).

## 🌍 Base URL

```text
https://api.aicord.cloud
```

## 🔐 Autenticacao

Use um dos formatos:

- `Authorization: Bearer <API_KEY>`
- `x-api-key: <API_KEY>`

## 🧩 Endpoints

| Metodo | Path | Descricao |
| --- | --- | --- |
| `GET` | `/` | Metadados do servico |
| `GET` | `/health` | Sinal de saude |
| `GET` | `/v1/models` | Aliases publicos |
| `POST` | `/v1/chat/completions` | Chat completion |

### Resposta de health

```json
{"ok": true}
```

## 🏷️ Headers importantes

- `x-aicord-tier`
- `x-aicord-model`
- `x-aicord-upstream`
- `x-aicord-latency-ms`
- `x-ratelimit-remaining`

## ❌ Formato de erro

```json
{
  "error": {
    "message": "Unauthorized",
    "type": "authentication_error"
  }
}
```

## 📄 OpenAPI

- `openapi/aicordcloud.openapi.yaml`

Validacao:

```bash
npx mintlify@latest openapi-check docs/openapi/aicordcloud.openapi.yaml
```
