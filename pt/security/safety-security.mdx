# 🛡️ Safety and Security (PT-BR)

> Baseline de seguranca para operar e vender AiCord.

## 🔐 Chaves

- Nunca versionar chave em Git
- Usar secret manager em producao
- Rotacionar chaves periodicamente

## 👮 Controle de acesso

- Menor privilegio para painel e servidor
- Comandos de gestao restritos a owner/admin

## ⚙️ Guardrails de runtime

- Timeout de request
- Rate limit por tenant
- Fila + backpressure

## 📜 Politica de logs

- Logar status, latencia, tier e rota upstream
- Nunca logar segredo completo
- Manter timeline para postmortem

## 📦 Licenciamento e distribuicao

- Escopo de suporte Paid explicito
- Restricoes de Trial documentadas
