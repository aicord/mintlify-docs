# ⚡ Quickstart (PT-BR)

> 🎯 Objetivo: sair do zero ate a primeira resposta valida em minutos.

## 1. 🔧 Configure o `.env` do bot

```env
AI_PROVIDER=openai
OPENAI_BASE_URL=https://api.aicord.cloud/v1
OPENAI_API_KEY=<chave_do_cliente>
OPENAI_MODEL=aicord/fast
AI_FALLBACK_PROVIDERS=
```

## 2. 💬 Configure os canais no Discord

1. `/chat user channel`
2. `/chat admin channel`
3. Opcional: `/chat prompt edit`

## 3. 📡 Valide a API

```bash
curl https://api.aicord.cloud/v1/chat/completions \
  -H "Authorization: Bearer SUA_CHAVE" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "aicord/fast",
    "messages": [{"role":"user","content":"Diga OK"}],
    "temperature": 0.2
  }'
```

✅ Esperado: `200 OK` com texto em `choices[0].message.content`.

## 4. ✅ Validacao no Discord

- Envie prompt curto no canal admin
- Verifique latencia e resposta
- Confira logs com provider e latencia

## 5. 🟢 Checklist de go-live

- [ ] Bot responde nos canais configurados
- [ ] `/doctor live_probe:true` aprovado
- [ ] `/panel` abre e persiste configuracoes
- [ ] Chave de API nao exposta em repositorio publico
- [ ] Caminho de fallback documentado
