# Runbook de Operacao (PT-BR)

## Rotina diaria

1. Health endpoint
2. Taxa de erro
3. p95 de latencia
4. Pressao de fila
5. Taxa de fallback

## Rotina semanal

1. Auditoria de rotacao de chave
2. Janela de update
3. Validacao de backup

## Incidentes

1. Classificar severidade
2. Registrar timeline
3. Mitigar primeiro
4. Fazer RCA apos estabilidade
5. Publicar resumo interno

## Politica de deploy

- PR obrigatorio
- Mudancas de producao com registro
- Plano de rollback antes do release
