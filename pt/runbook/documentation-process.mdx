# Processo de Documentacao (PT-BR)

## Fluxo

1. Abrir PR com escopo claro
2. Review tecnico
3. Review de linguagem
4. Merge apos aprovacoes

## Regras de consistencia

- Toda mudanca EN deve ter espelho PT no mesmo PR, ou issue vinculada com prazo.
- Exemplo de comando deve ser executavel.
- Variavel de ambiente citada deve existir no `.env.example`.
